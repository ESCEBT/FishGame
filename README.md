# FishGame
HTML5绘制的爱心鱼小游戏
打开html文件点击index直接运行即可体验此游戏；

##游戏规则
鼠标移动大鱼控制大鱼吃掉海藻，即可保持小鱼不死，若小鱼颜色变成白色，则小鱼死亡，游戏结束。
